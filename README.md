# worldbuilder
Worldbuilder Project in Java

Controls:
W - Move Up
A - Move Left
S - Move Down
D - Move Right
Shift - Move Faster
E - Toggle Inventory GUI
1-9 / Scroll Wheel - Block Selection
M - Mute Music
Z - Reset Camera
Ctrl + Scroll Up - Zoom In
Ctrl + Scroll Down - Zoom Out
Left Click - Destroy Block
Right Click - Place Block
